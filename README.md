# coreos-infra
A set of Ansible playbooks and net boot components to provide network services on a pair of CoreOS services.

For very small networks (or to model larger ones) 

## OS Config

* Butane
* Ignition

## Boot/Install

* DHCP
* TFTP

## Service Installation

* DNS - CoreDNS
* NTP - Chrony

* Monitoring - TBD
